Python 特点：优雅，明确，简单
Python适合的领域：1、Web网站和各种网络服务
		2、系统工具和脚本
		3、把其他语言开发的模块包装起来方便使用
Python不适合的领域：1、贴近硬件的开发
		   2、移动开发：IOS/Android有各自的开发语言（ObjC,Swift/Java）
		   3、游戏开发：C/C++
1、数据类型：整数，浮点数，字符串，布尔值
2、变量定义：python是一门动态语言，变量的类型不固定
3、字符串：（1）字符串可以用用  '' （当字符串中有"时）或 "" （当字符串中有'时）括起来，
如果两者都包含可以用转义字符 \ 进行转移
	   （2）如果字符串中有较多需转义，可在字符串前面加一个r，表示raw字符串，
里面的字符就不需要转义了
	   （3）但是 r' ' 不能表示多行字符，也不能表示含 ' 和 " 的字符串，
因为字符串的结尾会被误判，所以用 '''...''' 来表示有多行的字符串，如果需要转义，还可以在前面加上r
	   （4）中文字符串需要在第一行加注释 # -*- coding: utf-8 -*- 以告诉编译器用utf-8读取源代码
